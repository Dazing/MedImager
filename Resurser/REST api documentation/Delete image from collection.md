**Delete image from collection**
----
  Allows a user to delete an image from a collection

* **URL**

  /collection/{collectionID}

* **Method:**

  `DELETE`
  
*  **Header Params**

   **Required:**
 
   `Authorization: <token>` <br />

* **Data Params :**
  * **JSON:** <br />

```javascript
{       
        "collectionitemID": "123",
}
```
