**Delete collection**
----
  Allows a user to delete a collection

* **URL**

  /collection/

* **Method:**

  `DELETE`
  
*  **Header Params**

   **Required:**
 
   `Authorization: <token>` <br />

* **Data Params :**
  * **JSON:** <br />

```javascript
{       
        "collectionID": "12",
}
```
