**Add description to collection**
----
  Allows a user to add a description to a collection

* **URL**

  /collection/description

* **Method:**

  `PUT`
  
*  **Header Params**

   **Required:**
 
   `Authorization: <token>` <br />

* **Data Params :**
  * **JSON:** <br />

```javascript
{       
        "collectionDescr":"Interesting development",
        "collectionID": "12",
}
```
