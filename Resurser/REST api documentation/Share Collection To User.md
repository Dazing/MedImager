**Share collection to user**
----
  Allows a user to share a collection to another user

* **URL**

  /collection/share/{targetuserID}

* **Method:**

  `POST`
  
*  **Header Params**

   **Required:**
 
   `Authorization: <token>` <br />

* **Data Params :**

  * **JSON:** <br />

```javascript
{
        "collectionID": "12",
}
```
