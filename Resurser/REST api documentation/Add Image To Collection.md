**Add image to collection**
----
  Allows a user to add an image to a collection

* **URL**

  /collection/{collectionid}

* **Method:**

  `POST`
  
*  **Header Params**

   **Required:**
 
   `Authorization: <token>` <br />

* **Data Params :**

  * **JSON:** <br />

```javascript
{
        "examinationID": "22000300",
        "index": "2",
}
```
